Simple rest application with ExpressJS, VueJS, and MongoDB<br>
Not 100% funcionality, *just for learning*<br>

**Usage Server API**
* Enter api directory and run:<br>
    ```
    $ npm install
    ```
* Running in dev mode
    ```
    $ npm run start-dev
    ```

**Features**
* Server API
- [x] CRUD products
- [x] CRUD Users
- [x] JWT implemented
- [x] Some validations

* Client API
- [x] API comunication
- [x] User Register Page
- [x] Registry user
- [x] Login Page
- [ ] Persist token
- [ ] Login user in dashboard
- [ ] Client Dashboard
- [ ] Admin Dashboard

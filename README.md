Simple rest application with ExpressJS, VueJS, and MongoDB<br>
Not 100% funcionality, *just for learning*<br>

**Usage Server API**
* Enter api directory and run:<br>
    ```
    $ npm install
    ```
* Running in dev mode
    ```
    $ npm run start-dev
    ```

**Features**
* Server API
- [x] CRUD products
- [ ] CRUD Users
- [x] JWT implemented
- [ ] Some validations

* Client API
- [ ] API comunication
- [ ] Login Page
- [ ] User Dashboard
